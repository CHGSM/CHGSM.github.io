如何利用Hexo以及GitHub搭建自己的博客（高手勿进）

这是本人的第一篇博客，用于记录我是如何利用一些工具来搭建当前我所在的这个博客页面的一些过程以及一些注意事项，希望有意无意点进来看的各位能够通过我的这篇博客收获一些什么（至少可以安装成功吧）。
为了写这篇博客，我姑且狠一狠心，把我之前安装的东西全部卸载删除了。为了纪念一下我已经配置好的博客，先贴一张图来纪念一下，希望在这篇博客写完之后，我可以重新看到这个界面。

全部。。。。。。重新开始。

为了完成这个任务,我们需要做以下这些事情:

  1. 下载并安装Git(或者 Git Bash)
  2. 下载并安装Node.js
  3. 安装Hexo
  4. 注册GitHub账号(如果有就不必了),并生成SSH上传到GitHub上面
  5. 更改项目的一些配置
  6. 将配置好的项目部署上传到Github上面
  7. 对自己的博客做一些简单的配置文件修改
